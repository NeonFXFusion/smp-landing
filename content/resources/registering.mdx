## How to Register and Login on a Minecraft Server

Follow the prompts on the screen so make sure you have in game titles enabled!
Make sure you dont prefix any text with a command such as /register just directly type
the password in the chat.

1. Open in-game chat.
2. Type "[your chosen password]" to register.
3. Use "[your password]" to access the server.
